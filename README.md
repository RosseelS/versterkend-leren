### Git basics

1. **Installeer Git**

Download [Git](https://git-scm.com)

3. **Clone het project**

Open een terminal en voer uit:

   ```bash
   git clone LINK_VAN_HET_PROJECT
   ```

4. **Ga naar de projectmap**

   ```bash
   cd naam-van-het-project
   ```

---

### Wijzigingen binnenhalen (pull)

Voor je begint te werken haal je best eerst de laatste wijzigingen binnen:

```bash
git pull
```

Dit zorgt ervoor dat je **de nieuwste versie van het project** hebt.

---

### Wijzigingen opslaan en uploaden (push)

1. **Voeg je wijzigingen toe**

```bash
git add .
```

2. **Maak een commit**

```bash
git commit -m "beschrijving van je wijzigingen"
```

3. **Upload naar GitHub**

```bash
git push
```
